Скрипт поиска по сайту на основе Яндекс.XML.
php-MyAdmin.ru/learning/search.html (utf-8).

Условия добавления на сайт страницы поиска:
1) Сервер должен иметь выделенный IP, который необходимо зарегистрировать на странице http://xml.yandex.ru/settings.xml.
2) На странице отображения результатов поиска должна стоять ссылка на Яндекс.XML (http://xml.yandex.ru).

Конфигурирование скрипта - файл config.php:
1) Впишите в переменную $aLocal['ip'] ваш выделенный IP сервера.
    Пример:
    $aLocal['ip'] = '*.*.*.*';
2) Впишите в переменную $aLocal['url'] ваш адрес для совершения запроса.
    Пример:
    $aLocal['url'] = 'http://xmlsearch.yandex.ru/xmlsearch?user=name&key=********';
3) Впишите в переменную $aLocal['host'] домен вашего сайта.
    Пример:
    $aLocal['host'] = 'php-myadmin.ru';
4) Для проверки наличия в PHP обязательных расширений, а также корректности конфигурационных директив, запустите скрипт (search/test.php).

Добавление на сайт:
1) Для добавления формы поиска на страницу вашего сайта воспользуйтесь примером в файле (search/index.php).
2) Для добавления только поля поиска на любых других страницах вашего сайта, кроме страницы поиска, воспользуйтесь примером формы из файла (search/test.php).
3) Оформление осуществляется редактированием стилей в файле (search/search.css).
4) После установки скрипта удалите файлы search/test.php и search/readme.txt.
